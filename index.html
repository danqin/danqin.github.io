我读冯唐

+＋

我读冯唐的作品，字里行间透露着血腥。

+

一个人或是一群人，冯大厨拉过来观察几秒，然后一把刀一把斧子一顿乱砍，把人肉剁碎一地，再把胸腔里的大肠小肠腰子肚儿掏出来铺在书页上，告诉你人就是这几个脏器加一滩肉泥。

+

你捡起一块小腿骨，放在空荡荡的书包里，风急火燎的挤上早高峰的地铁，一手扶着天花板，一手拿着冯大厨切给你的还连着一根神经的小腿骨，垫起一只脚，拿出来胡乱一啃，要边笑变啃，眼睛要闭着，但要竖着耳朵听报站

+

放心，没有人会盯着你打量你在做什么，地铁上的人没有眼睛，他们本来是有的，一上车，就被手机吃掉了，就跟你现在吃着小腿骨，一样的吃法等到耳朵告诉你，该去地面上了，你就把吃剩下的扔了就好，或者继续赛回书包里，我一般不这么做，因为回到人间，我就不能再明目张胆的茹毛饮血了，倒不是因为公安局住着警察，因为太阳太大，我张不开嘴

+

你说冯唐是不是个血腥的混蛋？

回国到现在，我有五颜六色的感受，不过根据CMYK的颜色系统，加在一起后是黑色的墨水

+

我有强烈的挫败感，感觉一事无成，生活没有目标和方向，对周围的一切提不起兴趣，恐惧社交，不想花力气费心思，和周围的一切喘气的动物周折

+

然后慢慢开始怀疑是不是抑郁症，但我又害怕这种怀疑，我怕血液里真的有抑郁症的DNA

+

我一定要找出这一切不良心态的根源，我是对自己失望，是对工作，对感情，对父母，对朋友，对未来，对人生，对价值，对环境，我到底是对什么恐惧，不安，躁动，委屈，惶恐，聒噪，虚荣，面子，自由。。。到底是对什么有这种乱七八糟的想法

+

我看冯唐，坐地铁的时候看，看完了就想吐

+

我看红楼梦，睡前躺在床上看，看完了就想能坐个大观园的梦，梦见怡红院也好，梦见潇湘馆也好，我真想死在大观园里

+

我看博尔赫斯，下午写完字看，像是进了摩洛哥的YLS的大花园，里面种满了妖艳的食人花，散发着我几年前写给新概念作文比赛的参赛作品的香气，看完了胃里也是造了个蝴蝶窝一样，有种诡异的美好

+

我看通俗天文学，看了两页，觉得写的太浅就放下了，我是一个自以为是的人

+

我想找个人说说话，或者大哭一场，我怕我自己憋坏了，我又能找谁呢，我不能找zl，她不会想听的，她听我讲的太多，我不能找ph，我能说什么呢，她也讲了好多好多，我就点头，我是一个自以为是的人，我不能找。。。我没有人可以找，越长大越孤单，说的真是没错

+

我是不是不该回国，我又是个棉花耳朵，我是不是该第二次出国，美国么还是欧洲，或者澳洲，我不确定我出去后会痊愈，我出去后对母亲的牵挂怎么办

+

我现在到底是怎么了，是不是因为天太热了，还是不停的给自己找借口
